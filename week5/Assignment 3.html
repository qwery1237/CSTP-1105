Objects

Complete the following tasks. Name your html file Name_Assignment_3 and submit it before the deadline.
60% of the marks will go to successfully completing the tasks free of errors.
20% of the marks will go to how well you comment your code. This also incorporates good naming conventions for variables & proper formatting/organisation of your code.
20% of the marks will go to how efficient your code is using the methods you have learned up to this point.

  1. Detailed Country Objects
    • Create 3 objects. An object for America, France and UK.
    • Have these objects store the following properties: name, population, capital and isMonarchy (a boolean indicating whether the country has a monarchy or not). Assign the correct values for each of these properties.
    • Have these objects store a method called stateDetails which returns the following string

      "X is a country with a population of X and the capital city is X. This country is/is not a monarchy."

      Make sure to use "is" or "is not" to say whether the country is a monarchy or not.
    • Place these 3 objects in an array.
    • Create a function that takes in an array (the array of objects) as a parameter.
    • Have this function loop through the array and console.log the stateDetails() method from each object so that we see each country's description in the console.
  2. Counting Items in Huge Array
    The array below contains a bunch of strings in various amounts. Create an object to help you count each element in the array and in a single console.log, log the following:

    "A = ?
    B = ?
    C = ?
    ...
    Z = ?"

    where ? is the number of times that the letter appears in the array. Each letter that appears in the array should be present in this log. Finally, console.log "$ appears the most in the array at @ times" where $ is the most common letter and @ is the number of times it is present in the array.

    Read this part if you are having difficulty coming up with a solution. You have completed a question that resembles this in the In Class Tasks. However there are probably too many letters in this array for you to hard code them in as properties. Come up with a system that loops through the array, creates properties representing the letters if they are discovered for the first time, and increments the count on subsequent iterations in the loop when the letter is seen again. The object method hasOwnProperty() might be useful here.
